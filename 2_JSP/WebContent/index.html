* JSP(Java Server Page)

JSP란, 자바언어를 이용해서 HTML형식 내에서 JAVA소스코드를 사용할 수 있음(사용할 수 있는 기술)
서블릿에서 응답데이터를 구현했던 복잡함을 보다 간단하게 해결 가능한 기술
=> JSP의 가장 큰 장점은 Servlet에서는 비즈니스로직에만 집중하고
프레젠테이션 로직처리는(응답화면을 만들어내는 뷰) JSP에서 집중할 수 있게끔 분리하는 것

JSP 표현법

1. JSP 스크립팅 요소(Scripting Element)

JSP페이지에서 자바코드를 기술할 수 있게 해주는 기능

  1. 선언문 : <%! %>
    변수나 메서드를 선언하기 위해 사용
  2. 스크립틀릿 : &li;% %>
    JSP에서 자바코드를 기술하기 위한 가장 기본적인 표현법
  3. 표현식(출력식) : <%= %>
    자바에서 작성한 값(변수 또는 메서드)을 출력하기 위해 사용
